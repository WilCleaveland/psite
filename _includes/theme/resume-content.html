Point it at a live sales page: it reads the brand's voice, offer, and proof, extracts the mechanisms behind copy that already converted (paste a winner, screenshot one, or pull a competitor's currently-running Meta ads), generates a stress-tested matrix of fresh angles, then expands survivors into full emails or Meta ads. UGAR Campaign System

A Python pipeline that authors, audits, and pushes daily Maropost campaigns across multiple formats (Daily Style, Editorial PM, Daily 11am, and more). Dry-run mode for safe drafting; --execute for production push. Pre-flight auditor catches stale logos, body/text mismatches, broken affiliate links, and missing required elements before any send.

Tools: Python · Maropost API · Claude Code

Daily Hard-Bounce Suppression

Scheduled, unattended task that runs every day: discovers all sent campaigns in the window, pulls bounce reports via the Maropost API, identifies hard bounces and mailbox-full SMTP codes, and updates the active suppression list. Reduces ESP costs by removing bad addresses before the next send and protects sender reputation.

Tools: Python · Maropost API · macOS launchd / Claude scheduled tasks

Click-Tracking Slug Infrastructure

Custom redirect domain (mail.preparedgunowners.com/<slug>) that replaces long affiliate URLs in campaign body with branded short links. Server-side PHP redirector reads a client-uploaded mapping file and forwards to the affiliate URL with all tracking parameters intact (aff_sub, click_id, etc.). Reduces body clutter, improves deliverability, and gives the brand consistent click-tracking surface.

Tools: PHP redirector · JSON mapping · Python author-side integration · Claude Code

Cross-Platform API Integrations

In-progress: a Konnektive CRM ⇄ Ottertext SMS tag-mapping layer. Customer segments defined in Konnektive (purchase history, tags, lifecycle stage) flow into Ottertext campaigns automatically — removing the manual list-pull step from the SMS send workflow. Foundation for unifying email + SMS audience targeting under one CRM-driven source of truth.

Tools: Python · Konnektive API · Ottertext API · Claude Code
